# Regional Roadmap Draft

## Recommended swim lanes

Use the y-axis for solution type and keep bureau or portfolio as a color or label on each bar.

1. Domestic Budget and Execution
2. Overseas Dashboard and Budget Formulation
3. Country or Bureau Specific Solutions
4. Appropriations and Spend Plan
5. Regional Platform and Shared Services

The first four align to the swim-lane concept already started in the PDF. The fifth lane is needed so Portal, DT&A, Apptio, OM and Support, and other regional foundation work do not get forced into the wrong category.

## Visual design guidance

Show one bar per task in the appropriate swim lane. Put the bureau or portfolio in the bar label and use color for bureau family:

- WHA: one color
- NEA: one color
- SCA: one color
- CARE or WCF: one color
- Regional All or Planning: neutral color

Use milestone diamonds above the bars so quarter labels remain readable.

## How to show staff without cluttering the roadmap

Do not create separate staff swim lanes on the same chart. That will make the roadmap unreadable.

Use this instead:

1. Put small owner tags at the right end of each bar, such as `MF`, `SF`, `IW`.
2. Limit bar labels to one to three names. If more people support the work, use a small badge like `+3`.
3. Keep leadership off the task bars. Show Nupur and Vinson only as milestone sponsors in a thin top row.
4. Add a side legend called `Staffing Overlay` that translates initials to full names and role types.
5. If you need a second visual, make a one-page companion heatmap by quarter showing where each person is overloaded.
